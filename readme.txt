Adds a sidebar widget to display your recently reviewed books at LibraryThing.

== Description ==

Uses lastRSS (http://lastrss.oslab.net/) for parsing a LibraryThing (http://www.librarything.com) user review XML file
for display in a sidebar widget.

== Installation ==


1. Upload the 'libraryThing' director to your blog in the wp-content/plugins/ directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure the widget in the 'Design'/'Appearance' menu in Wordpress.
